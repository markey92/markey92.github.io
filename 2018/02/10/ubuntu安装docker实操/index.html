ubuntu安装docker实操

发表于 2018-02-10 | 分类于 日常知识点 | | 阅读次数:

安装前检查,查看Ubuntu版本

Docker 要求 Ubuntu 系统的内核版本高于 3.10

  • uname -a
  • lsb_release -a

开始安装docker

  • apt-get install docker.io

检查安装后的docker版本

  • docker -v
  • docker version

尝试运行hello-world

  • docker run hello-world

卸载docker

  • apt-get remove docker-engine
# docker
Hello World
